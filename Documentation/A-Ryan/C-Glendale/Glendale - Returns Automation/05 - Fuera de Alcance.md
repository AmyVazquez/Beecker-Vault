# Fuera de Alcance

Los siguientes elementos están **explícitamente fuera de alcance** para esta fase del proyecto:

| Elemento | Razón |
|---|---|
| **RMA / Exchange Automation** | Requiere demasiadas decisiones manuales (identificación de SKU, negociación de precio, revisión de serviceabilidad) |
| **WorldShip Desktop Application** | Reemplazado completamente por UPS Web API; no se hará automatización de UI sobre la aplicación desktop de WorldShip |
| **Integración UI con MOM ERP** | Reemplazado por integración backend |
| **Returns de canales distintos al formulario web** | Todos los returns deben ingresar por el formulario web; los returns históricos no enviados por el formulario quedan fuera de alcance |

> Cualquier elemento fuera de este alcance que se requiera en el futuro deberá tratarse como un **change request**.
